Verilen benzer bir düzenle basit bir web sayfası oluşturun. CSS’in temel özelliklerini uygulayın. Sadece HTML5 + CSS3 kullanın, hazır framework kullanmak yasak[cite: 1, 10].

İçerik

Bu alanda örnek içerik metinleri bulunmaktadır. Css özellikleriyle bu metinler arasında boşluklar (line-height) ve hizalamalar (text-align) düzenlerleriz[cite: 7].

Önce CSS ile kutuları / alanları yan yana getirmeyi, ardından alt alta yerleştirmeyi öğrendik[cite: 6]. Şimdi ise CSS seçicilerini kullanacağız.

Yan Menü / Bilgi

Bu alan yan menü veya bilgi kısmıdır. CSS ile genişlik (width), yükseklik (height), kenar boşlukları (margin/padding) ve kenarlık (border) özelliklerini kullanarak düzenleyebiliriz[cite: 6].

Öğrendiklerimizi hiyerarşik ve grup seçicilerle uygulayıp farklı bölümlere stil verebiliriz[cite: 9].
